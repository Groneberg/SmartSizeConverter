## English Version

### 🌟 Foreword & Market Potential
In 2026, online shopping is the global standard, yet one massive problem remains unsolved: **Size Chaos.** Over 70% of fashion e-commerce returns are due to fit issues. This costs retailers billions in lost revenue, frustrates customers, and puts a significant strain on the environment through unnecessary logistics and carbon emissions.

**Perfect Fit** addresses this head-on. Instead of just displaying static charts, this app serves as a personal shopping companion. Leveraging the power of the **Isar Database**, it stores local profiles, accounts for brand-specific deviations (e.g., Nike vs. Adidas), and tells the user in seconds which size they *actually* need.

### 🚀 Vision
A minimalist, lightning-fast, and privacy-focused solution that ends the guesswork in fashion retail — **Offline-first & User-centered.**

### 🗺️ Roadmap
* **Phase 1: The Foundation (MVP)**
    * [ ] Setup: Flutter project structure with Isar DB integration.
    * [ ] Data Engine: Core conversion logic for international shoe sizes (EU, US, UK, JP).
    * [ ] Core UI: Intuitive "Quick-Converter" screen.
* **Phase 2: Personalization**
    * [ ] User Profiles: Local storage of physical measurements (cm/inch).
    * [ ] Brand Overrides: Save specific brand experiences (e.g., "Size up for Adidas").
* **Phase 3: Intelligence**
    * [ ] Smart Recommendations: Automated suggestions based on measurements.
    * [ ] Community Insights: Crowdsourced "True to Size" data.

### ⚙️ Installation (Dev-Quickstart)
1.  **Clone the repository**
2.  **Install dependencies:**
    ```bash
    flutter pub get
    ```
3.  **Run Isar code generator:**
    ```bash
    dart run build_runner build
    ```
4.  **Run App:**
    ```bash
    flutter run
    ```

## Deutsche Version

### 🌟 Vorwort & Marktpotenzial
Im Jahr 2026 ist Online-Shopping der Standard, doch ein Problem bleibt ungelöst: **Das Größen-Chaos.** Über 70 % der Retouren im Modebereich entstehen durch Passform-Probleme. Das kostet Händler Milliarden, frustriert Kunden und belastet durch unnötige Logistik die Umwelt.

**Perfect Fit** setzt genau hier an. Anstatt nur statische Tabellen anzuzeigen, ist diese App ein persönlicher Begleiter, der mithilfe der **Isar Database** lokale Profile speichert, Markenunterschiede berücksichtigt und dem Nutzer in Sekunden sagt, welche Größe er *wirklich* braucht.

### 🚀 Vision
Eine minimalistische, blitzschnelle und datenschutzfreundliche Lösung, die das Rätselraten beim Kleidungskauf beendet – **Offline-first & User-centered.**

### 🗺️ Roadmap
* **Phase 1: Das Fundament (MVP)**
    * [ ] Setup: Flutter Projektstruktur mit Isar DB Integration.
    * [ ] Data Engine: Basis-Umrechnungslogik für internationale Schuhgrößen.
    * [ ] Core UI: Intuitive "Quick-Converter" Ansicht.
* **Phase 2: Personalisierung**
    * [ ] User Profiles: Speichern von Körpermaßen direkt auf dem Gerät.
    * [ ] Brand Overrides: Spezifische Erfahrungen pro Marke speichern.
* **Phase 3: Intelligence**
    * [ ] Smart Recommendations: Automatische Vorschläge basierend auf Maßen.
    * [ ] Community Insights: Crowdsourcing von Passform-Erfahrungen.
